# 🎨 1강: 브랜딩 & 상담 준비

> **핵심 메시지:** 브랜딩은 로고가 아니라 **'한 문장'**이다.
> **소요 시간:** 40분 | **난이도:** ⭐ (입문)

---

## 🎯 이 강의에서 만드는 것

| # | 산출물 | 설명 |
|---|---|---|
| 1 | 브랜드 브리프 | 학원의 정체성을 담은 1페이지 문서 |
| 2 | 슬로건 5종 | 학원을 한 문장으로 표현하는 슬로건 |
| 3 | FAQ 10개 | 학부모가 가장 많이 묻는 질문과 답변 |
| 4 | 상담 A4 | 상담 시 보여줄 학원 소개 자료 |
| 5 | 팜플렛 문구 | 배포용 팜플렛에 들어갈 카피 |

---

## 📚 사전 준비

1. [Google Gemini](https://gemini.google.com/) 접속 → 로그인
2. 아래 **학원 정보**를 미리 정리해두세요:

```
학원명: _______________
위치: _______________
대상 학생: _______________  (예: 초등 3~6학년)
주요 프로그램: _______________
학원의 강점: _______________
원장님 교육 철학: _______________
```

---

## 🔬 실습 1: 브랜드 브리프 생성

### RTF+C 프롬프트

아래 프롬프트를 Gemini에 **그대로 복사-붙여넣기** 하세요.
`[괄호]` 안의 내용만 여러분의 학원 정보로 바꿔주세요.

```
너는 교육 브랜딩 10년 경력의 전략 컨설턴트야. (R)

아래 정보를 바탕으로 우리 학원의 브랜드 브리프를 작성해줘. (T)

학원명: [학원명]
위치: [위치]
대상: [대상 학생]
주요 프로그램: [주요 프로그램]
강점: [학원의 강점]
교육 철학: [원장님의 교육 철학]

다음 항목을 포함하는 1페이지 분량의 브랜드 브리프를 작성해줘:
1. 미션 (한 문장)
2. 비전 (한 문장)
3. 핵심 가치 3가지
4. 타겟 학부모 페르소나 (이름, 나이, 고민)
5. 브랜드 톤 & 매너 (3가지 키워드)
6. 경쟁 학원과의 차별점 (F)

과장하지 말고 실현 가능한 약속만 담아줘.
학부모 불안을 자극하는 표현은 사용하지 마. (C)
```

### ✅ 체크포인트

- [ ] 미션이 한 문장으로 명확한가?
- [ ] 타겟 페르소나가 구체적인가?
- [ ] 과장된 표현이 없는가?

### 💡 팁

> 결과가 마음에 들지 않으면 **"미션을 더 간결하게 줄여줘"** 또는 **"타겟 학부모의 나이를 35-42세로 좁혀줘"** 처럼 후속 지시를 내려보세요. 이것을 **반복 프롬프팅(Iterative Prompting)**이라고 합니다.

## 🔬 실습 2: 슬로건 5종 생성

### RTF+C 프롬프트

```
너는 카피라이터 겸 교육 마케팅 전문가야. (R)

아래 브랜드 브리프를 바탕으로 학원 슬로건 5개를 만들어줘. (T)

[위에서 생성한 브랜드 브리프를 여기에 붙여넣기]

조건:
- 각 슬로건은 10자 이내
- 학부모에게 신뢰감을 주는 톤
- 운율이나 대구법 활용
- 표 형태로 정리 (번호 | 슬로건 | 컨셉 키워드) (F)

"1등", "최고", "무조건" 같은 과장 표현 금지.
"불안", "걱정", "뒤처진다" 같은 공포 마케팅 표현 금지. (C)
```

### ✅ 체크포인트

- [ ] 10자 이내인가?
- [ ] 과장/공포 표현이 없는가?
- [ ] 5개 중 마음에 드는 것이 있는가?

### 💡 팁

> 마음에 드는 슬로건이 있다면 **"2번 슬로건을 기반으로 변형 3개 더 만들어줘"** 라고 요청하세요.

## 🔬 실습 3: FAQ 10개 생성

### RTF+C 프롬프트

```
너는 학원 상담 경력 15년의 베테랑 상담 실장이야. (R)

[학원명]에 대해 학부모가 가장 많이 하는 질문 10개와
답변을 작성해줘. (T)

학원 정보:
- [위에서 생성한 브랜드 브리프를 여기에 붙여넣기]

형식:
Q1. [질문]
A1. [답변 3줄 이내]
...
Q10. [질문]
A10. [답변 3줄 이내] (F)

답변은 불안을 해소하는 톤으로 작성해줘.
약속할 수 없는 성적 향상 보장은 하지 마.
각 답변은 3줄 이내로 간결하게. (C)
```

### ✅ 체크포인트

- [ ] 질문이 실제 학부모가 할 법한 것인가?
- [ ] 답변이 불안 해소 톤인가?
- [ ] 성적 보장 같은 위험한 약속이 없는가?

## 🔬 실습 4: 상담 A4 자료 생성

### RTF+C 프롬프트

```
너는 학원 운영 컨설턴트이자 디자인 디렉터야. (R)

상담 시 학부모에게 보여줄 A4 1장짜리 학원 소개 자료의
텍스트 콘텐츠를 작성해줘. (T)

포함할 내용:
1. 학원 한 줄 소개 (슬로건)
2. 교육 철학 (3줄)
3. 주요 프로그램 목록 (표)
4. 수업 시간표 요약
5. 학부모 후기 (가상 2개)
6. 연락처 및 위치

학원 정보:
[브랜드 브리프 붙여넣기]
선택한 슬로건: [선택한 슬로건] (F)

실제 학부모 후기가 아닌 가상 후기임을 명시해줘.
A4 1장에 들어갈 분량으로 제한해줘. (C)
```

## 🔬 실습 5: 팜플렛 문구 생성

### RTF+C 프롬프트

```
너는 학원 홍보 전문 카피라이터야. (R)

3단 접지 팜플렛에 들어갈 문구를 작성해줘. (T)

구성:
- 앞면: 슬로건 + 한 줄 소개
- 안쪽 왼쪽: 교육 철학 + 강점 3가지
- 안쪽 가운데: 프로그램 소개 (표)
- 안쪽 오른쪽: 학부모 후기 + CTA
- 뒷면: 위치, 연락처, QR코드 자리

[브랜드 브리프 + 슬로건 붙여넣기]

각 영역은 **헤딩 + 본문** 형태로 구분해줘. (F)

각 영역별 글자 수를 50자 이내로 제한해줘.
"지금 바로 전화하세요!" 같은 강압적 CTA는 피해줘. (C)
```
