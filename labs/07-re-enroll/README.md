## 📖 핵심 개념: 퇴원 유형 5가지

| 유형 | 사유 | 재등록 가능성 | 접근법 |
|---|---|---|---|
| 🅰️ 성적 불만 | "효과가 없어요" | ★★★ 높음 | 구체적 성과 데이터 제시 |
| 🅱️ 경제적 이유 | "학원비가 부담돼요" | ★★☆ 중간 | 유연한 옵션 제안 |
| 🅲️ 이사/전학 | "멀어져서요" | ★☆☆ 낮음 | 온라인 옵션 안내 |
| 🅳️ 다른 학원 | "다른 곳을 다녀보려고요" | ★★★ 높음 | 3개월 후 재접촉 |
| 🅴️ 학생 의지 | "아이가 싫어해요" | ★★☆ 중간 | 프로그램 변경 제안 |

## 🔬 실습 1: 퇴원 유형 분류표 & 대응 전략

### RTF+C 프롬프트

```
너는 학원 경영 컨설턴트이자 고객 유지(Retention) 전문가야.
학원 퇴원 관리 시스템을 10년간 설계해왔어. (R)

학원 퇴원 유형 5가지에 대한 상세 분류표를 만들어줘. (T)

각 유형별로:
1. 유형명
2. 대표 퇴원 사유 (학부모 실제 말투)
3. 숨겨진 진짜 이유
4. 재등록 가능성 (상/중/하)
5. 최적 재접촉 타이밍
6. 대응 전략 핵심 1줄

5가지 유형을 표 형태로 정리해줘. (F)

학부모가 말하는 이유와 진짜 이유가 다를 수 있음을 반영.
퇴원 자체를 부정적으로 프레이밍하지 마.
"학부모의 선택을 존중하되, 문은 항상 열려있다" 톤으로. (C)
```

## 🔬 실습 2: 유형별 재접촉 메시지 (3단계)

### RTF+C 프롬프트

```
너는 학원 CRM 메시지 전문 카피라이터야. (R)

퇴원 유형별 3단계 재접촉 메시지를 만들어줘. (T)

5가지 유형:
A. 성적 불만형
B. 경제적 이유형
C. 이사/전학형
D. 다른 학원형
E. 학생 의지형

각 유형별 3단계:
- 1단계 (D+3): 감사 & 안부 메시지
- 2단계 (D+14): 근황 확인 & 정보 제공
- 3단계 (D+30): 복귀 기회 안내

각 메시지는 카톡 형태, 5줄 이내.
총 15개 메시지 (5유형 × 3단계) (F)

"돌아오세요" 같은 직접적 권유 금지.
1단계는 순수한 감사만, 2단계부터 넌지시.
학부모 자존심을 건드리지 않는 톤.
이모지 적절히 사용. (C)
```

### ✅ 체크포인트 (유형 A — 성적 불만형 예시)

**1단계 (D+3) 감사:**
```
안녕하세요, 민준이 어머니 😊
그동안 저희 학원을 믿고 맡겨주셔서 감사합니다.
민준이가 보여준 성장이 앞으로도 이어지길 바랍니다.
항상 응원하겠습니다! 💪
```

**2단계 (D+14) 근황:**
```
안녕하세요, 민준이 어머니 🙂
혹시 민준이 영어 학습은 잘 이어가고 있나요?
최근 저희가 도입한 AI 성적 분석 시스템이
학생들 성적 향상에 큰 도움이 되고 있어요 📈
궁금하신 점이 있으시면 편하게 연락주세요!
```

**3단계 (D+30) 기회:**
```
안녕하세요, 민준이 어머니 ☀️
다음 달 새 학기 프로그램을 준비하며
기존 학생분들께 우선 안내드리고 있어요.
관심 있으시면 체험 수업도 가능합니다 😊
편한 시간에 연락주세요!
```

## 🔬 실습 3: 30일 캠페인 시트

### RTF+C 프롬프트

```
너는 마케팅 자동화 전문가야. (R)

퇴원 후 30일간의 재등록 캠페인 일정표를 만들어줘.
Google Sheets에 바로 넣을 수 있는 형태로. (T)

표 구조:
| D+X | 날짜(예시) | 유형 | 채널 | 메시지 키워드 | 담당 | 완료 |
|---|---|---|---|---|---|---|

30일 일정:
- D+1: 퇴원 사유 기록
- D+3: 1차 감사 메시지 (카톡)
- D+7: 학부모 SNS 활동 체크
- D+10: 주변 학부모 통해 근황 파악
- D+14: 2차 근황 메시지 (카톡)
- D+21: 학원 소식/이벤트 공유
- D+25: 학원 성과 사례 공유
- D+30: 3차 복귀 기회 메시지

나머지 날짜도 적절히 채워줘. (F)

매일 접촉은 과하니 적절한 간격으로.
압박감 없는 자연스러운 접촉 위주.
최대 카톡 6회, 전화 2회 이내. (C)
```

## 🔬 실습 4: 복귀 혜택 제안서

### RTF+C 프롬프트

```
너는 학원 경영 전문가야. (R)

퇴원생 복귀를 위한 혜택 패키지 3종을 설계해줘. (T)

패키지 구성:
1. 🥉 라이트 패키지 — 부담 없는 시작
2. 🥈 스탠다드 패키지 — 가장 인기
3. 🥇 프리미엄 패키지 — 최대 혜택

각 패키지:
- 패키지명
- 포함 내용
- 할인/혜택 내용
- 조건 (기간 등)
- 일반 등록과의 차이

표 형태로 한눈에 비교 가능하게. (F)

학원 수익성을 해치지 않는 선에서.
"특별 혜택"이지 "할인 투매"가 아닌 포지셔닝.
기존 학생(재원생)이 보면 불공평하게 느끼지 않을 수준. (C)
```
