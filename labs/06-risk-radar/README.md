## 🔬 실습 1: 성적 추이 시트 만들기

### RTF+C 프롬프트

```
너는 Google Sheets 전문가이자 교육 데이터 분석가야. (R)

학원 성적 관리 스프레드시트를 만들기 위한
Google Sheets 수식과 차트 설정을 알려줘. (T)

현재 데이터 구조:
- A열: 학생 이름
- B~G열: 3월~8월 점수
- 1행: 헤더

필요한 것들:
1. H열: 평균 점수 수식
2. I열: 최고점 수식
3. J열: 최저점 수식
4. K열: 변화량 (마지막 달 - 첫 달)
5. L열: 추세 (상승/유지/하락)

각 수식을 정확한 셀 참조와 함께 제공해줘.
차트 만드는 방법도 단계별로. (F)

모든 수식은 2행부터 시작한다고 가정.
Google Sheets 호환 수식만 사용 (Excel 전용 함수 금지). (C)
```

### 단계별 실행

1. H2에 평균 수식 입력: `=AVERAGE(B2:G2)`
2. I2에 최고점: `=MAX(B2:G2)`
3. J2에 최저점: `=MIN(B2:G2)`
4. K2에 변화량: `=G2-B2`
5. L2에 추세: `=IF(K2>5,"📈 상승",IF(K2<-5,"📉 하락","➡️ 유지"))`
6. **모든 수식을 아래로 드래그!**

## 🔬 실습 2: 위험 감지 수식

### 핵심 개념: 3회 연속 하락 = 위험 🚨

```
4월<3월 AND 5월<4월 AND 6월<5월 → 위험!
```

### RTF+C 프롬프트

```
너는 학원 데이터 분석 전문가야. (R)

학생의 최근 3개월 연속 하락을 감지하는
Google Sheets 수식을 만들어줘. (T)

데이터 위치:
- B~G열에 3월~8월 점수
- M열에 위험 감지 결과를 넣을 예정

조건:
- 최근 3개월 연속 하락 → "🚨 위험"
- 최근 2개월 연속 하락 → "⚠️ 주의"
- 그 외 → "✅ 정상"

수식은 G열(8월)을 기준으로 역방향 3개월 체크.
셀 참조를 정확하게 제공해줘. (F)

Google Sheets 호환 수식만.
AND/IF 함수 조합으로. (C)
```

### 예상 수식 (M2에 입력)

```
=IF(AND(G2<F2, F2<E2, E2<D2), "🚨 위험",
  IF(AND(G2<F2, F2<E2), "⚠️ 주의", "✅ 정상"))
```

## 🔬 실습 3: 조건부 서식 대시보드

### 단계별 가이드

1. **전체 점수 영역(B2:G6)** 선택
2. **서식** → **조건부 서식**
3. 규칙 추가:

| 조건 | 색상 | 의미 |
|---|---|---|
| 90 이상 | 💚 초록 | 우수 |
| 70~89 | 💛 노란 | 양호 |
| 50~69 | 🧡 주황 | 노력 필요 |
| 49 이하 | ❤️ 빨강 | 위험 |

4. **M열(위험 감지)** 에도 조건부 서식:

| 값 | 색상 |
|---|---|
| 🚨 위험 | 빨간 배경 |
| ⚠️ 주의 | 노란 배경 |
| ✅ 정상 | 초록 배경 |

## 🔬 실습 4: 개입 메시지 3종

### RTF+C 프롬프트

```
너는 학원 상담 전문가이자 위기 대응 커뮤니케이터야. (R)

위험 수준별 학부모 개입 메시지 3종을 만들어줘. (T)

1. 🚨 위험 (3회 연속 하락) — 긴급 상담 요청
2. ⚠️ 주의 (2회 연속 하락) — 주의 안내
3. 📊 일반 (정기 보고) — 월간 성적 안내

각 메시지:
- 카톡 메시지 형태 (5줄 이내)
- 불안을 유발하지 않는 톤
- 구체적 다음 단계 제안 (F)

"걱정하셔야 합니다" 금지.
위험 메시지도 "함께 방법을 찾아보겠습니다" 톤으로.
학부모가 읽고 분노하지 않을 톤으로. (C)
```
