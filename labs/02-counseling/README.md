## 🔬 실습 1: 학부모 유형 분류 & 브리핑 생성

### 학부모 4유형 이해하기

| 유형 | 특징 | 상담 전략 |
|---|---|---|
| 🅰️ 성적 집착형 | "몇 점 오르나요?" | 구체적 데이터 + 단계별 계획 제시 |
| 🅱️ 비교 불안형 | "옆집 아이는..." | 개별 성장 그래프 + 비교 대신 변화 강조 |
| 🅲️ 방관 위탁형 | "알아서 해주세요" | 학부모 역할 가이드 + 주간 소통 약속 |
| 🅳️ 정보 수집형 | "커리큘럼 자세히요" | 체계적 자료 + 시범 수업 제안 |

### RTF+C 프롬프트

```
너는 영어학원 상담 20년 경력의 원장이자 상담 코치야. (R)

아래 학원 정보를 기반으로, 학부모 4가지 유형별
3분 상담 브리핑 대본을 만들어줘. (T)

학원 정보:
[1강에서 만든 브랜드 브리프 붙여넣기]

학부모 유형:
A. 성적 집착형 — "몇 점 오르나요?"
B. 비교 불안형 — "옆집 아이는..."
C. 방관 위탁형 — "알아서 해주세요"
D. 정보 수집형 — "커리큘럼 자세히요"

각 유형별 대본 구성:
1. 오프닝 (인사 + 공감 1줄)
2. 핵심 메시지 (학원 강점 연결 2줄)
3. 구체적 제안 (수업/프로그램 1가지)
4. 클로징 (다음 단계 안내 1줄)

표 형태로 유형별로 정리해줘. (F)

"무조건 성적이 오른다" 같은 약속 금지.
각 대본은 실제 3분 안에 말할 수 있는 분량으로.
학부모의 감정을 먼저 인정하는 톤으로 작성해줘. (C)
```

### ✅ 체크포인트

- [ ] 4유형 모두 생성되었는가?
- [ ] 각 대본이 3분 분량인가?
- [ ] 감정 인정 → 제안 순서가 맞는가?

## 🔬 실습 2: Gemini Gems 2.0 설정

### Gems란?

> **Gems**는 Gemini의 맞춤 챗봇 기능입니다.
> 학원 정보를 미리 넣어두면, 언제든 학원 맞춤 답변을 받을 수 있습니다.

### 설정 프롬프트

```
너는 영어학원 상담 전문 AI야. (R)

Gemini Gems 2.0에 넣을 시스템 프롬프트를 생성해줘. (T)

이 Gem의 역할:
- 학부모 상담 시뮬레이터
- 학부모 유형 자동 감지
- 유형에 맞는 상담 대본 실시간 제안
- 상담 후 기록 자동 정리

학원 정보:
[브랜드 브리프 붙여넣기]

Gem 시스템 프롬프트 형식으로 작성해줘:
---
이름: [학원명] 상담 파트너
역할: ...
지침: ...
제약: ...
--- (F)

답변 톤은 항상 불안 해소형으로.
학원의 실제 인원, 시설 등 모르는 정보는 추측하지 마. (C)
```

### Gems 설정 방법

1. [Gemini](https://gemini.google.com/) → 왼쪽 메뉴 **Gem Manager**
2. **+ New Gem** 클릭
3. 위에서 생성된 시스템 프롬프트를 **Instructions** 에 붙여넣기
4. **Save** → 이제 '상담 파트너' Gem이 생성됩니다!

## 🔬 실습 3: 상담 기록 템플릿 생성

### RTF+C 프롬프트

```
너는 학원 CRM 시스템 설계 전문가야. (R)

Google Sheets에서 사용할 상담 기록 템플릿의
열(Column) 구조를 설계해줘. (T)

포함할 정보:
- 상담일시
- 학생명
- 학부모명
- 학부모 연락처
- 학부모 유형 (A/B/C/D)
- 현재 수준
- 학부모 주요 관심사
- 상담 내용 요약
- 제안한 프로그램
- 다음 액션
- 상담 결과 (등록/보류/미등록)
- 팔로업 날짜

Google Sheets에 바로 넣을 수 있는 표 형태로 정리해줘.
첫 행은 열 제목, 둘째 행은 예시 데이터. (F)

개인정보보호법을 고려하여 민감 정보 최소화.
열 수는 15개 이하로 제한. (C)
```

### Google Sheets에 적용하기

1. [Google Sheets](https://sheets.google.com/) → 새 스프레드시트
2. 이름: `[학원명] 상담 기록`
3. 생성된 열 구조를 첫 행에 붙여넣기
4. 조건부 서식: 상담 결과가 "미등록"이면 빨간색으로!

## 🔬 실습 4: 반박 대응 스크립트

### RTF+C 프롬프트

```
너는 설득 커뮤니케이션 전문가이자 학원 상담 코치야. (R)

학부모가 상담 중 자주 하는 반박 5가지와
각각에 대한 대응 화법을 작성해줘. (T)

반박 예시:
1. "수학이 먼저 아닌가요?"
2. "학원이 너무 많은데..."
3. "효과가 있는지 모르겠어요"
4. "아이가 학원 가기 싫어해요"
5. "온라인으로도 되지 않나요?"

각 반박에 대해:
- 학부모 감정 인정 (1줄)
- 관점 전환 (1줄)
- 우리 학원 연결 (1줄)
- 제안 (1줄) (F)

절대 학부모를 설득하려 하지 마.
"듣겠습니다" → "제안드려도 될까요?" 톤으로. (C)
```

### ✅ 체크포인트

- [ ] 5가지 반박 모두 대응이 있는가?
- [ ] "설득"이 아닌 "제안" 톤인가?
- [ ] 실제 상담에서 자연스러운가?
