# 📢 8강: 마케팅 자동화

> **핵심 메시지:** 마케팅은 **AIDA 한 사이클**이면 충분하다.
> **소요 시간:** 40분 | **난이도:** ⭐⭐ (초급)

---

## 🎯 이 강의에서 만드는 것

| # | 산출물 | 설명 |
|---|---|---|
| 1 | AIDA 블로그 글 | 주의→흥미→욕구→행동 구조의 블로그 글 |
| 2 | 5채널 카피 | 블로그/인스타/네이버/카톡/전단지 변환 |
| 3 | 포스터 문구 | 수강생 모집 포스터 텍스트 |
| 4 | 시즌별 마케팅 캘린더 | 연간 마케팅 이벤트 계획 |

---

## 📖 핵심 개념: AIDA 프레임워크

```
A — Attention  (주의)     "이런 적 있으세요?"
I — Interest   (흥미)     "이렇게 하면 달라집니다"
D — Desire     (욕구)     "실제로 이런 결과가 나왔습니다"
A — Action     (행동)     "지금 체험 수업을 신청하세요"
```

---

## 🔬 실습 1: AIDA 블로그 글 작성

### RTF+C 프롬프트

```
너는 학원 마케팅 전문 카피라이터야.
네이버 블로그가 학원 마케팅의 70%를 차지한다는 걸 잘 알아. (R)

AIDA 구조에 맞는 학원 홍보 블로그 글을 작성해줘. (T)

학원 정보:
[1강에서 만든 브랜드 브리프 붙여넣기]

글 구조:
1. 📣 Attention (제목 + 도입 3줄)
   - 학부모 공감 질문으로 시작
2. 💡 Interest (본문 5줄)
   - 우리 학원만의 차별점 설명
3. ❤️ Desire (사례 3줄)
   - 변화 사례 (가상이라도 자연스럽게)
4. 🚀 Action (마무리 2줄)
   - 체험 수업 안내 + 연락처

총 400-600자, 네이버 블로그에 최적화. (F)

네이버 SEO를 위해 핵심 키워드 3개를 자연스럽게 포함.
(예: "[지역명] 영어학원", "초등 영어", "레벨테스트")
"1등 학원", "최고" 같은 과장 금지.
가상 사례는 너무 극적이지 않게. (C)
```

### 💡 네이버 블로그 SEO 팁

> **제목에 키워드 넣기:** "강남 초등영어 | 레벨테스트 무료 체험"
> **태그 활용:** 지역명, 학원 종류, 학년, 프로그램명
> **사진 3장 이상:** 학원 외관, 수업 모습, 교재 사진

## 🔬 실습 2: 5채널 카피 변환

### RTF+C 프롬프트

```
너는 멀티채널 마케팅 전문가야. (R)

위에서 작성한 AIDA 블로그 글을
5가지 채널에 맞게 변환해줘. (T)

변환 대상:
1. 📝 네이버 블로그 (이미 완성) — 요약 버전 200자
2. 📸 인스타그램 — 이미지 위 텍스트 3줄 + 해시태그 10개
3. 📍 네이버 플레이스 소식 — 100자 이내 공지
4. 💬 카카오톡 단체 메시지 — 5줄 이내
5. 📄 전단지 — 앞면 헤드라인 + 서브카피

각 채널별 형식을 명확히 구분해서 제공해줘. (F)

각 채널의 글자 수 제한을 지켜줘.
인스타 해시태그는 관련성 높은 것으로.
카톡은 링크 포함 가능하게.
전단지는 10m 거리에서 읽을 수 있는 크기 고려. (C)
```

## 🔬 실습 3: 수강생 모집 포스터 문구

### RTF+C 프롬프트

```
너는 학원 홍보 디자이너 겸 카피라이터야. (R)

수강생 모집 포스터에 들어갈 텍스트를 만들어줘. (T)

포스터 구성:
1. 📢 헤드라인 (10자 이내, 큰 글씨)
2. 📋 서브 헤드라인 (20자 이내)
3. 📌 핵심 혜택 3가지 (불릿 포인트)
4. 🎁 이벤트/혜택 (있다면)
5. 📞 연락처 + 위치
6. 🔗 QR코드 자리 표시

학원 정보: [브랜드 브리프] (F)

포스터는 3초 안에 핵심이 전달되어야 해.
글씨가 너무 많으면 안 됨 — 각 항목 최소한으로.
CTA(Call to Action)는 1개만. (C)
```

### 💡 Canva로 포스터 만들기

> 1. [Canva](https://www.canva.com/) 접속
> 2. **포스터** 검색 → 템플릿 선택
> 3. 위에서 만든 문구를 텍스트에 입력
> 4. 학원 사진, 로고 추가
> 5. 다운로드 → 인쇄 or SNS 게시

## 🔬 실습 4: 연간 마케팅 캘린더

### RTF+C 프롬프트

```
너는 학원 마케팅 전략가야. (R)

영어학원의 연간 마케팅 이벤트 캘린더를 만들어줘. (T)

4시즌 구분:
🌸 봄 (3-5월): 신학기 모집 시즌
☀️ 여름 (6-8월): 여름방학 특강 시즌
🍂 가을 (9-11월): 2학기 안정화 시즌
❄️ 겨울 (12-2월): 겨울방학 + 신년 모집 시즌

각 시즌별:
- 핵심 마케팅 목표
- 추천 이벤트 2-3개
- 핵심 메시지 1줄
- 사용 채널

표 형태: 월 | 이벤트 | 목표 | 채널 | 예산등급 (F)

학원 규모를 고려해 저예산 위주.
온/오프라인 균형있게.
각 이벤트는 1-2주 내에 실행 가능한 것으로. (C)
```
