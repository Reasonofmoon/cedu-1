## 📖 핵심 개념: 4시즌 프레임워크

```
🌸 봄 (3-5월) — ACQUIRE: 신규 모집 집중
☀️ 여름 (6-8월) — INTENSIFY: 방학 특강 + 실력 점프
🍂 가을 (9-11월) — RETAIN: 유지 & 내신 관리
❄️ 겨울 (12-2월) — PLAN: 결산 & 차년도 설계
```

## 🔬 실습 1: 4시즌 전략표 생성

### RTF+C 프롬프트

```
너는 학원 경영 전략 컨설턴트 20년 경력이야.
4시즌 학원 운영 전략에 특화돼 있어. (R)

영어학원의 4시즌 연간 전략표를 만들어줘. (T)

학원 정보: [브랜드 브리프]
학원 규모: [현재 학생 수]명
목표 학생 수: [목표]명

각 시즌별:
1. 시즌명 + 기간
2. 핵심 목표 (1줄)
3. 전략 3가지
4. 핵심 이벤트 2개
5. 주의할 점 1가지
6. 성공 지표 (KPI) 2개

표 형태로 4시즌 한눈에 비교 가능하게. (F)

학원 규모에 맞는 현실적 전략으로.
대형 프랜차이즈가 아닌 개인 학원 기준.
비용 대비 효과(ROI)가 높은 전략 위주. (C)
```

## 🔬 실습 2: 월별 운영 계획

### RTF+C 프롬프트

```
너는 학원 운영 매니저 전문가야. (R)

1월부터 12월까지의 월별 운영 계획표를 만들어줘. (T)

각 월별:
| 월 | 시즌 | 핵심 이벤트 | 마케팅 | 학사 | 재무 |
|---|---|---|---|---|---|

월별 포함 정보:
- 행사/이벤트 (학교 일정 연동)
- 마케팅 활동
- 학사 일정 (시험, 레벨테스트 등)
- 재무 체크 (등록 마감, 특강비 등) (F)

한국 학교 일정(학기, 방학, 시험)에 맞춰줘.
2026년 기준 실제 일정 반영.
각 월의 핵심 한 줄 슬로건도 추가. (C)
```

## 🔬 실습 3: KPI 10개 설정

### RTF+C 프롬프트

```
너는 학원 경영 데이터 분석 전문가야. (R)

학원 운영의 핵심 KPI(핵심 성과 지표) 10개를 설정하고
추적할 수 있는 Google Sheets 구조를 만들어줘. (T)

KPI 후보:
1. 월 신규 등록 수
2. 월 퇴원 수
3. 재원생 유지율
4. 평균 재원 기간 (개월)
5. 레벨 상승 비율
6. 학부모 만족도 점수
7. 수업 출석률
8. 숙제 완료율
9. 레벨테스트 평균 점수
10. 마케팅 전환율 (문의→등록)

Google Sheets 구조:
- 시트1: KPI 대시보드 (게이지 수식)
- 시트2: 월간 데이터 입력

각 KPI에 대해:
목표값 | 현재값 | 달성률 수식 | 상태(🟢🟡🔴) (F)

수동으로 입력 가능한 간단한 구조로.
매주 10분이면 업데이트할 수 있도록.
달성률 자동 계산 수식 포함. (C)
```

### Sheets 수식 예시

```
달성률: =현재값/목표값*100
상태: =IF(달성률>=90,"🟢",IF(달성률>=70,"🟡","🔴"))
```

## 🔬 실습 4: 주간 루틴 템플릿

### RTF+C 프롬프트

```
너는 학원장 시간 관리 코치야. (R)

학원 원장의 주간 루틴 템플릿을 만들어줘.
반복 업무를 요일별로 정리해. (T)

요일별 구조:
| 요일 | 오전 루틴 | 수업 시간 | 오후 루틴 | 관리 업무 |
|---|---|---|---|---|

포함할 업무:
- 상담 (주 2-3회)
- 강사 미팅 (주 1회)
- 마케팅 콘텐츠 (주 1회)
- KPI 업데이트 (주 1회)
- 학부모 소통 (매일)
- 수업 관찰 (주 1회)
- 재무 점검 (주 1회) (F)

1인 원장 기준 (혼자 운영).
수업 시간 제외 관리 업무 시간은 하루 2시간 이내.
주말은 최소한의 업무만. (C)
```
