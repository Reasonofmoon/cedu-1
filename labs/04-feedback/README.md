# 💬 4강: AI 피드백 시스템

> **핵심 메시지:** 피드백은 "잘했다/못했다"가 아니라 **성장 내러티브**다.
> **소요 시간:** 40분 | **난이도:** ⭐⭐ (초급)

---

## 🎯 이 강의에서 만드는 것

| # | 산출물 | 설명 |
|---|---|---|
| 1 | 학생 피드백 | 학생에게 직접 전달하는 격려형 코멘트 |
| 2 | 학부모 리포트 | 학부모에게 보내는 성장 분석 리포트 |
| 3 | 카톡 메시지 | 카카오톡으로 보내는 간결한 피드백 |
| 4 | 4주 전략 | 향후 4주 학습 전략 제안 |

---

## 📚 사전 준비

학생 1명의 **테스트 결과 데이터**를 준비하세요 (가상도 OK):

```
학생명: _______________
학년: _______________
레벨:  _______________  (A반/B반/C반)
이번 점수: _____ / 100
지난달 점수: _____ / 100
강한 영역: _______________  (예: 어휘, 독해)
약한 영역: _______________  (예: 문법, 쓰기)
수업 태도: _______________  (적극적/보통/소극적)
```

---

## 📖 핵심 개념: 형성적 평가 프레이밍

### ❌ 일반적 피드백 (판단형)
```
"문법이 약합니다. 더 열심히 해야 합니다."
```

### ✅ 형성적 피드백 (성장형)
```
"어휘력이 지난달 대비 15% 성장했어요!
문법은 현재 기초를 다지는 단계인데,
'be동사 정리' 워크시트 3장이면 다음 달엔 큰 변화가 올 거예요."
```

**차이점:** 현재 위치 → 성장한 부분 → 다음 단계 구체적 제안

## 🔬 실습 1: 학생 피드백 생성

### RTF+C 프롬프트

```
너는 10년 경력의 영어학원 원장이자 교육 심리 전문가야.
학생의 자기효능감을 높이는 피드백 작성에 특화되어 있어. (R)

아래 학생 정보를 바탕으로 학생에게 직접 전달할
성장형 피드백을 작성해줘. (T)

학생 정보:
학생명: [학생명]
학년: [학년]
레벨: [레벨]
이번 점수: [점수]/100
지난달 점수: [지난달 점수]/100
강한 영역: [강한 영역]
약한 영역: [약한 영역]
수업 태도: [수업 태도]

피드백 구성:
1. 🎉 칭찬 (성장한 부분 구체적으로)
2. 💪 강점 확인 (잘하는 영역 언급)
3. 🌱 성장 포인트 (약한 영역을 긍정적으로)
4. 🎯 이번 주 도전 과제 (구체적 1가지) (F)

"못한다", "부족하다" 대신 "성장 중이다", "곧 달라질 거야" 사용.
학생이 직접 읽는다고 가정하고 반말 톤으로.
5줄 이내로 간결하게. (C)
```

### ✅ 체크포인트

- [ ] 칭찬이 구체적인가? (점수 변화 언급)
- [ ] 부정적 표현이 없는가?
- [ ] 도전 과제가 실행 가능한가?

## 🔬 실습 2: 학부모 리포트 생성

### RTF+C 프롬프트

```
너는 학원 운영 컨설턴트이자 교육 분석 전문가야. (R)

아래 학생 정보를 바탕으로 학부모에게 전달할
월간 성장 리포트를 작성해줘. (T)

학생 정보:
[위와 동일한 학생 정보]

리포트 구성:
1. 📊 이번 달 성적 요약 (점수 + 변화량)
2. 📈 성장 트렌드 (3개월 추이 설명 — 데이터 없으면 이번 달 기준)
3. 🏆 이번 달 최고 성과 (구체적 1가지)
4. 📋 영역별 분석 (표 형태: 영역 | 수준 | 변화 | 코멘트)
5. 🗓️ 다음 달 학습 계획 (3가지 구체적 목표)
6. 💡 가정 학습 가이드 (학부모가 할 수 있는 것 2가지)

전문적이지만 이해하기 쉬운 존댓말로. (F)

"걱정하셔야 합니다" 같은 불안 유발 표현 금지.
객관적 데이터 중심, 학부모 역할 가이드 포함.
A4 1장 분량. (C)
```

## 🔬 실습 3: 카톡 피드백 메시지

### RTF+C 프롬프트

```
너는 학부모 소통 전문 상담사야. (R)

학부모에게 카카오톡으로 보내는 간결한 주간 피드백
메시지를 작성해줘. (T)

학생 정보:
[학생 정보 간략히]

형식:
---
안녕하세요, [학생명] 학부모님 😊
[인사 1줄]

📊 이번 주 결과
[핵심 1줄]

🌟 칭찬 포인트
[1줄]

📝 이번 주 과제
[1줄]

감사합니다. [학원명] 드림
---

카톡에서 읽기 편한 길이로. (F)

이모지 적절히 사용.
전체 5줄 이내.
"상담 원하시면 연락주세요" 같은 CTA 1개만. (C)
```

### ✅ 체크포인트

- [ ] 5줄 이내인가?
- [ ] 카톡에서 스크롤 없이 읽히는가?
- [ ] 이모지가 과하지 않은가?

## 🔬 실습 4: 4주 학습 전략 생성

### RTF+C 프롬프트

```
너는 영어 학습 설계 전문가이자 개인 과외 전략가야. (R)

아래 학생의 약점을 보완하기 위한
4주 집중 학습 전략을 만들어줘. (T)

학생 정보:
[학생 정보]
보완 필요 영역: [약한 영역]

4주 전략 구성 (표 형태):
| 주차 | 목표 | 활동 | 교재/자료 | 확인 방법 |
|---|---|---|---|---|
| 1주 | ... | ... | ... | ... |
| 2주 | ... | ... | ... | ... |
| 3주 | ... | ... | ... | ... |
| 4주 | ... | ... | ... | ... | (F)

각 주차의 활동은 학원 수업 시간 내에 가능한 것으로.
가정 학습은 하루 15분 이내로.
4주 후 측정 가능한 목표를 설정해줘. (C)
```

## 🔬 실습 5: 배치 피드백 (여러 학생 한번에)

여러 학생의 피드백을 한번에 만들어봅시다!

### RTF+C 프롬프트

```
너는 학원 피드백 자동화 전문가야. (R)

아래 5명의 학생 데이터를 보고
각각에 대한 학생 피드백 + 카톡 메시지를 만들어줘. (T)

학생 데이터:
| 이름 | 학년 | 점수 | 지난달 | 강점 | 약점 |
|---|---|---|---|---|---|
| 김민준 | 초4 | 78 | 65 | 어휘 | 문법 |
| 이서연 | 초5 | 92 | 88 | 독해 | 쓰기 |
| 박지호 | 초3 | 45 | 50 | 듣기 | 어휘 |
| 최수아 | 초6 | 85 | 85 | 문법 | 독해 |
| 정예준 | 초4 | 60 | 55 | 쓰기 | 문법 |

각 학생별로:
[학생 피드백] + [카톡 메시지] 세트로 작성. (F)

5명 모두 동일한 형식으로.
점수가 하락한 학생은 더 격려하는 톤으로. (C)
```
