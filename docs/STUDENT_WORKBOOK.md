각 강의는 60분 기준이다.

01. 브랜딩 & 상담 준비

오늘의 목표: 학원의 정체성을 한 문장으로 정리한다.

오늘의 도구: ChatGPT, Gemini app / AI Studio

오늘의 산출물: 브랜드 브리프, 슬로건 3종

[ ] 로그인할 계정이 준비되었다

[ ] 오늘 사용할 예시 자료 또는 내 자료가 준비되었다

[ ] 결과물을 저장할 폴더나 문서 위치를 정했다

[ ] 도구 사용 시 개인정보와 기관 정보 범위를 확인했다

이 주제와 관련해 지금 가장 오래 걸리는 일은 무엇인가?

오늘 결과물을 실제 어디에 바로 써 볼 수 있는가?

도구 사용 전에 먼저 정리해야 할 정보는 무엇인가?

도구 소개: ChatGPT: Projects에 학원 정보와 슬로건 시안을 누적하면 차시 간 맥락 관리가 쉽다. Gemini: 앱에서 빠르게 문구 초안을 만들고, AI Studio에서는 표 형식 출력으로 다듬기 좋다.

오늘 내가 메인으로 사용할 도구:

대체 도구 또는 백업 방법:

오늘 시연을 보면서 확인할 기능 1가지:

오늘 내가 넣을 핵심 입력값 1:

오늘 내가 넣을 핵심 입력값 2:

오늘 내가 넣을 핵심 입력값 3:

기본 예시로 첫 결과물을 만든다.

내 상황에 맞게 입력값을 바꿔 다시 만든다.

결과물의 문장, 형식, 대상, 목적을 점검한다.

실제 공유 또는 사용 가능한 수준으로 수정한다.

최종본을 저장하고 파일명 또는 문서명을 정리한다.

중간 점검 체크리스트

[ ] 목표가 분명한가

[ ] 내 데이터나 내 상황이 반영되었는가

[ ] 결과물이 실제로 바로 쓸 수 있는 형식인가

[ ] 사람이 마지막 검수를 할 수 있게 되어 있는가

[ ] 다음 차시 입력값으로 다시 활용할 수 있는가
