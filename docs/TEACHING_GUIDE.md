CEDU-1 Teaching Guide

기준일: 2026-03-07

대상: AI를 처음 실무에 적용하는 영어학원 원장, 부원장, 팀장

수업 길이: 강의당 60분

과정 목표: 반복 문서를 AI에게 위임하고, 학원 운영 텍스트를 표준화한다

60분 흐름: 도입 5분 -> 개념 10분 -> 도구 소개/시연 10분 -> 실습 25분 -> 공유 7분 -> 정리 3분

기준이며, 실제 사용 가능 기능은 플랜과 조직 정책에 따라 달라질 수 있다.

01. 브랜딩 & 상담 준비

목표: 학원의 정체성을 한 문장으로 정리한다.

Gemini app / AI Studio

: Projects에 학원 정보와 슬로건 시안을 누적하면 차시 간 맥락 관리가 쉽다.

: 앱에서 빠르게 문구 초안을 만들고, AI Studio에서는 표 형식 출력으로 다듬기 좋다.

도입 5분: 현재 사용하는 학원 소개 문구 점검

개념 10분: RTF+C와 브랜드 브리프 구조 설명

도구/시연 10분: ChatGPT와 Gemini로 슬로건 비교 생성

실습 25분: 브랜드 브리프, 슬로건 3종, FAQ 5문항 작성

공유 7분: 추상적 문구와 구체적 문구 비교

정리 3분: 최종 문장 1개 확정

산출물: 브랜드 브리프, 슬로건 3종

오프닝: "오늘 목표는 멋있는 말이 아니라 학부모가 바로 이해하는 한 문장을 만드는 것입니다."

전개: "ChatGPT는 문장 확장에, Gemini는 빠른 비교 초안에 강하니 두 도구의 차이를 같이 보겠습니다."

마무리: "지금 만든 한 문장이 이후 상담과 마케팅 문서의 기준이 됩니다."

02. 상담 브리핑 자동화

목표: 상담 질문 흐름과 학부모 응답 스크립트를 표준화한다.

: 긴 상담 문장을 자연스럽게 정리하고 artifact 형태로 대본을 분리하기 좋다.

: 교재 소개서나 학원 안내 자료를 넣고 FAQ를 만들기 좋다.

도입 5분: 상담에서 자주 막히는 질문 수집

개념 10분: 탐색-진단-제안-확약 구조 설명

도구/시연 10분: Claude로 상담 대본, NotebookLM으로 FAQ 생성

실습 25분: 학년별 상담 스크립트와 상담 요약 템플릿 작성

공유 7분: 닫힌 질문과 열린 질문 비교

정리 3분: 다음 차시에 연결할 레벨테스트 설명문 1개 저장

산출물: 상담 질문 시퀀스, FAQ, 스크립트

오프닝: "상담을 잘하는 사람은 말을 많이 하는 사람이 아니라 질문 순서를 설계한 사람입니다."

전개: "Claude는 문체를 다듬고, NotebookLM은 자료 기반 FAQ를 만드는 데 강합니다."

마무리: "오늘 만든 질문 흐름이 있으면 상담 품질 편차가 크게 줄어듭니다."

정리 3분: 약점 태그 3개 확정

전개: "Gemini는 구조화 초안에, ChatGPT는 설명 문장에 각각 강점이 있습니다."

마무리: "오늘 만든 분류 기준이 이후 피드백과 반편성 설명의 기준이 됩니다."

04. AI 피드백 시스템

공유 7분: 칭찬만 있는 피드백과 행동 중심 피드백 비교

오프닝: "좋은 피드백은 예쁘게 쓰는 문장이 아니라 다음 행동이 보이는 문장입니다."

전개: "ChatGPT로 속도를 확보하고 Claude로 톤을 다듬는 조합이 실무에서 안정적입니다."

마무리: "오늘 만든 구조는 학부모 안내문과 시험 대비 코멘트로 바로 재사용할 수 있습니다."

전개: "Gemini로 내용 초안을 만들고 Canva로 즉시 배포 가능한 형식까지 연결해 보겠습니다."

마무리: "패키지 형식만 잡히면 다음 시험에도 재사용 속도가 빨라집니다."
